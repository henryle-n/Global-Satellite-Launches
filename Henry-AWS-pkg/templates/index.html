Project Description

In this project, a Comma Seperated Values (CSV) Satellite Lauches dated from 1974-2020 was downloaded. Dataset contains 2,666 satellites currenly orbiting the Earth, which includes country of origin, purpose, and other operational details of these satellites. By data visualization, the following data exploration can be made: thoroughly.

  • Satellite distributions among nations: who are the top 10 Satellite Owners?
  • How many Satellite Launches in a particular day, month, and year?
  • Interactive Data Table returning data based on user's query inputs.

Besides visualization, many more data exploration can be achieved by:

  • When was the oldest working satellite launched?
  • At what altitudes (orbiting class) do most satellites orbit?
  • What activities are most satellites involved with?
  • And many more ...
